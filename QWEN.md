# StarGatesTheGame — Проект документации

## Обзор проекта

**StarGatesTheGame** — это проект игры в жанре научно-фантастического приключения с элементами RPG. Проект включает документацию и веб-приложение для просмотра всей информации об игре.

### sgdocs — Документационный портал

Веб-приложение для документации, построенное на современном стеке:

- **Фреймворк:** React 19.2.4
- **Язык:** TypeScript 5.9.3
- **Сборщик:** Vite 8.0.1
- **Роутинг:** React Router 7
- **Компилятор:** React Compiler (через babel-plugin-react-compiler)

| Документация | `/docs` | Руководство разработчика, AI инструкции, спецификации |
| Roadmap | `/roadmap` | Этапы разработки, milestones, прогресс |
| Devlog | `/devlog` | История разработки, идеи, решения, заметки |
| Команда | `/team` | Участники проекта, роли, контакты |

## Сборка и запуск

Все команды выполняются из директории `sgdocs/`:

| Команда | Описание |
|---------|----------|
| `npm install` | Установка зависимостей |
| `npm run dev` | Запуск dev-сервера с HMR |
| `npm run build` | Сборка для продакшена |
| `npm run preview` | Предпросмотр продакшен-сборки |
| `npm run lint` | Запуск ESLint |

## Конфигурация

### Vite (`vite.config.ts`)

- Включён плагин `@vitejs/plugin-react`
- Настроен Babel с пресетом React Compiler
- Настроены path aliases для удобных импортов

### TypeScript

Проект использует solution-style tsconfig с раздельными конфигурациями для:
- Приложения (`tsconfig.app.json`)
- Node-окружения (`tsconfig.node.json`)

### Типы данных

Все типы определены в `src/types/index.ts` и включают:
- Навигация
- Концепт-арты и галерея
- Лор (миры, история, фракции, персонажи)
- Игровые механики
- Архитектура проекта
- Документация
- Roadmap
- Devlog
- Команда

## Архитектурные принципы

1. **Данные отделены от компонентов** — все данные хранятся в `src/data/*.ts`
2. **Строгая типизация** — все данные типизированы через TypeScript интерфейсы
3. **Модульность** — компоненты разделены на layout и UI
4. **Низкая связанность** — компоненты переиспользуемы
5. **Адаптивность** — базовая поддержка мобильных устройств
