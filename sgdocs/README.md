# Star Gates — Документация проекта

Веб-приложение для документации игры Star Gates, построенное на React + TypeScript + Vite.

## Структура проекта

```
sgdocs/
├── src/
│   ├── components/
│   │   ├── layout/          # Layout-компоненты (Header, Footer, Layout)
│   │   └── ui/              # Переиспользуемые UI компоненты
│   ├── data/                # Типизированные данные проекта
│   ├── pages/               # Страницы приложения
│   ├── styles/              # Глобальные стили
│   ├── types/               # TypeScript типы и интерфейсы
│   ├── App.tsx              # Главный компонент с роутингом
│   └── main.tsx             # Точка входа
├── public/
│   └── assets/              # Статические ресурсы (изображения)
└── package.json
```

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview

# Линтинг
npm run lint
```

## Страницы

| Страница | Путь | Описание |
|----------|------|----------|
| Главная | `/` | Hero секция, превью игры, концепт-арты |
| Об игре | `/overview` | Полное описание, жанр, особенности |
| Лор | `/lore` | Миры, история, фракции, персонажи |
| Механики | `/mechanics` | Core loop, бой, прогрессия, экономика |
| Архитектура | `/architecture` | Для разработчиков: модули, потоки данных |
| Документация | `/docs` | Руководства, AI инструкции, спецификации |
| Roadmap | `/roadmap` | Этапы разработки, milestones, статусы |
| Devlog | `/devlog` | История разработки, заметки, решения |
| Команда | `/team` | Участники проекта, роли |

## Архитектура данных

Все данные хранятся в отдельных файлах в папке `src/data/`:

- `navigation.ts` — навигация
- `home.ts` — данные главной страницы
- `overview.ts` — описание игры
- `lore.ts` — лор (миры, история, фракции, персонажи)
- `mechanics.ts` — игровые механики
- `architecture.ts` — архитектура проекта
- `documentation.ts` — документация
- `roadmap.ts` — roadmap разработки
- `devlog.ts` — записи devlog
- `team.ts` — команда

Данные агрегируются в `src/data/index.ts` для удобного импорта.

## Компоненты

### Layout
- `Header` — шапка с навигацией
- `Footer` — подвал с ссылками
- `Layout` — обёрка для страниц

### UI
- `Button` — кнопки с вариантами
- `Card` — карточки контента
- `Modal` — модальные окна
- `Gallery` — галерея с lightbox
- `Section` — секции страниц

## Стили

- CSS Variables для темизации
- Адаптивный дизайн (mobile-first)
- Тёмная тема с фиолетовыми акцентами

## Технологии

- React 19
- TypeScript 5.9
- Vite 8
- React Router 7
- Babel с React Compiler
